# Code of Conduct

Chronicle adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

Please read the full text at the link above. The short version:

- **Be kind.** Assume good intent.
- **Be respectful.** Disagree without attacking the person.
- **Be inclusive.** Chronicle welcomes contributors of every background, skill level, and perspective.
- **No harassment, no discrimination, no personal attacks.** Period.

## Scope

This Code of Conduct applies to all project spaces — GitHub issues, pull requests, discussions, Discord, and any other official Chronicle venue — and to public spaces when an individual represents the project.

## Reporting

If you experience or witness unacceptable behavior, report it privately to **conduct@chronicle.sh**.

Reports are reviewed and acted on within 7 days. Reporter identity is kept confidential by default. Retaliation against reporters is itself a violation.

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards. Maintainers who do not follow or enforce the Code of Conduct in good faith may face temporary or permanent consequences determined by other members of the project leadership.
