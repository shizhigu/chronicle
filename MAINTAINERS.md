## Areas

- **Core types / primitives** — `packages/core/`
- **Engine + rule enforcement** — `packages/engine/`
- **Agent runtime (pi-agent integration)** — `packages/runtime/`
- **Natural-language compiler** — `packages/compiler/`
- **CLI** — `packages/cli/`
- **Dashboard / rendering** — `packages/dashboard/`
- **Docs / ADRs / governance** — `docs/`, `docs/adr/`, this file, `CONTRIBUTING.md`
- **Infra / CI / releases** — `.github/`, `lefthook.yml`, `biome.json`

The `.github/CODEOWNERS` file maps directories to the handles responsible — keep the two in sync.
